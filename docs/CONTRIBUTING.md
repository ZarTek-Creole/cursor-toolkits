# Contributing

Merci de contribuer à cursor-toolkits !

## Guidelines

### Code Style

- Respecter les conventions par stack
- Documenter les ajouts
- Tester vos modifications

### Templates

- Suivre structure existante
- Respecter token budgets (<500 lignes)
- Inclure tests de base

### Documentation

- Mettre à jour README si nécessaire
- Ajouter exemples d'usage
- Documenter nouvelles features

## Tests

```bash
make test
make validate
```

## Pull Requests

1. Fork le repository
2. Créer une branche
3. Faire vos modifications
4. Lancer les tests
5. Soumettre PR avec description
